To run the project:
@Prerequisites
First install NodeJs: can be downloaded here: https://nodejs.org/en/

Open terminal and then run: " npm install -g cordova ionic " to install cordova and ionic.

ionic framework: http://ionicframework.com
cordova: https://cordova.apache.org

@running the project

To run the project
Open terminal, move to this project directory and the run: "ionic serve"
